version 1.0.0.0

Effects works on mobile / PC / consoles with vertexlit / forward / deferred renderer and dx9, dx11, openGL.
All effects optimized for mobile and pc. So you can use this effects even on old mobiles (for example with mali-400, about 50-60 fps). 


You can easy change a color of effect. (In editor or runtime). Just add the script "Colorizer" on prefab effect and change "TintColor" property.
If you want to change the scale of effect, just add the script "ParticleSystemScaler" on prefab effect and change "scale" property.
